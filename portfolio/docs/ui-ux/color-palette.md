# Philosophy

The portfolio follows a modern, premium developer aesthetic inspired by professional SaaS products.

Design goals:

- Minimal
- High contrast
- Accessible
- Professional
- Technology-focused
- Not gaming/neon

# Accessibility

Minimum contrast ratio

4.5:1

Large text

3:1

Interactive elements must always have visible hover and focus states.

---

# Usage Rules

Primary Blue

- CTA Buttons
- Active Navigation
- Links
- Icons

Secondary Blue

- Hover States
- Highlights

Surface

- Cards
- Panels

Background

- Entire application

Avoid introducing additional accent colors unless approved.
